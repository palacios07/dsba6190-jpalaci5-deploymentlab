## Getting Started with Terraform

Make your `main.tf`, `backend.tf`, and `variables.tf` files.

Initialize Terraform Locally
```bash
terraform init
```

Check your HCL formatting
```bash
terraform fmt --recursive
```

See what Terraform will deploy
```bash
terraform plan
```
